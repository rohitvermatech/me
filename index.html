My Story

I Started With Code.
Now I Ship Products.

I started out as a backend developer and chatbot builder. The kind of person who lived in the terminal, wrote APIs from scratch, and loved making systems talk to each other. That foundation never left me.

Over the years I picked up one skill after another. API integrations. WordPress sites. Web apps. DevOps. Power Automate workflows. n8n automation pipelines. Frontend interfaces. I taught myself each layer of the stack until I could take an idea from zero to deployed, without handing it off to anyone.

Then AI showed up and everything got faster. I started using it to write frontend code I'd never touched before. Backend I already owned. DevOps I'd learned. Suddenly I could ship real, functional, deployed products at a speed that wasn't possible before. Some people call it "vibe coding." I call it building without limits.

I don't get stuck. I don't say "that's not my area." If something needs to be built, whether it's a chatbot, an automation pipeline, a web app, an AI agent, or a full product, I scope it, design it, and ship it.

How I Got Here

The Journey

2017 - 2019

Where It Started: Chatbots & Backend

Began my career building chatbot systems and backend APIs. This is where I fell in love with automation logic, making systems respond intelligently to user input. Worked across small dev shops and internships, writing real production code from day one.

2019 - 2022

Going Full Stack: Web, APIs & Products

Expanded into full stack development. WordPress sites, web apps, backend systems, and increasingly complex API integrations. Started doing freelance work alongside employment. Learned that clients don't care about your stack. They care if the thing works.

2022 - 2025

Enterprise & Integration Consulting

Stepped into a consulting role, working with enterprise clients on digital transformation and API integration strategy. Got deep into designing systems that connect complex, fragmented business tools.

2025 - Now

The AI Era: Vibe Coding & Building Everything

AI changed the game. Building automation agents, deploying RAG bots for clients, and moving into fully AI-assisted product development. Now I can build what used to take a team. Solo, fast, and production-ready.
