My Story

Backend Roots.
Full Stack Reality.

I started writing backend code and building chatbots 7+ years ago. APIs, webhooks, server logic, database design. That was my world. I didn't just use APIs. I built them, connected them, broke them apart, and rewired them into systems that actually worked in production.

From there I kept going. I learned how to deploy what I built. Docker, Linux servers, cloud hosting. Then I picked up frontend. Then RPA and workflow automation with n8n and Power Automate. I never stopped at one layer because I wanted to own the entire pipeline, from the first API call to the final deployed product. No dependencies on other teams.

Now I build with AI in the loop. I use LLMs to accelerate frontend, generate automation logic, and prototype faster than most teams. I connect AI APIs from OpenAI, Gemini, Claude, and open-source models into real products. RAG pipelines, agentic workflows, auto-response systems, content automation. This is what vibe coding actually looks like: you understand the full stack deeply enough that AI becomes a multiplier, not a crutch. I'm not prompting and praying. I'm engineering with AI as a tool in my stack.

If a system has an API, I can plug into it. If it doesn't, I'll build the bridge. CRMs, messaging platforms, databases, third-party SaaS tools, internal systems. I've connected them all. And if the whole thing needs to be built from scratch, I do that too.
